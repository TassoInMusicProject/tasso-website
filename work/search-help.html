Search help

For text searches, type any full or partial word to highlight words in the score. Phrases are not currently allowed. Music searches are independent of text searches: there are three features that can be mixed and matched:

Pitches a  through  g Diatonic pitch names (any chromatic inflection).
n, #, - Optional chromatic quality (natural, sharp, flat).
r Rest.
Interval 1 Repeated note
2, -2 Up a second or down a second.
3, -3 Up third or down a third.
8, -8 Up or down an octave.
Rhythm    0, 1, 2, 4, 8 breve, whole, half, quarter, eighth.
. augmentation dot (e.g., 2. 4)

Example music searches

Spaces are usually optional in the search fields, as demonstrated in the following examples. Searches in separate features can also be done at the same time, such as searching for C, D, E, F, G, that starts with a whole note on C, by adding "1" in the rhythm search box.
cdefg (in pitch search box) Diatonic pitch classes C, D, E, F, and G in sequence with any duration. (spaces between notes are optional). This will also match to C, D, E-flat, F, G, since any chromatic inflection is allowed for each note.
cndne-fngn (in pitch search box) Search for C-natural, D-natural, E-flat, F-natural, and G-natural in sequence.
2-31 (in interval search box) up a step, then down a third then repeat the pitch.
1442 (in rhythm search box) Whole note (or rest), then two quarter notes then a half note.
2c4def1g (in pitch search box) Half-note C, quarter note D, E with any rhythm, F, whole-note G. Pitches and rhythms can be interleaved in pitch searches. Place the rhythm before the pitch. Do not put spaces between a rhythm and its pitch, but spaces between notes are optional. If a note does not have a rhythm, then it will match to any rhythm.
ceg / 33 (in pitch and interval search boxes) Search for C, E, and G in sequence, with the requirement that the E is a third above the C and the G is a third above the E.
